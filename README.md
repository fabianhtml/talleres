# Cosa del Futuro

Generador de prompts creativos basado en [The Thing from the Future](http://situationlab.org/project/the-thing-from-the-future/) de Situation Lab.

## Demo

[talleres.fablab.blog/the-thing-from-the-future](https://talleres.fablab.blog/the-thing-from-the-future/)

## Qué es

Cada prompt combina cinco dimensiones:

- **Tiempo**: horizonte temporal (mañana, dentro de una década, dentro de una generación, etc.)
- **Arco**: tipo de futuro (crecimiento, colapso, disciplina, transformación)
- **Terreno**: área temática (salud, educación, trabajo, etc.)
- **Objeto**: tipo de artefacto (dispositivo, prenda, ritual, etc.)
- **Emoción**: sensación que evoca (asombro, nostalgia, miedo, etc.)

Úsalo para sesiones de diseño especulativo, talleres de innovación, ejercicios creativos o simplemente para dejar volar la imaginación.
